# Pillar E — Interactive / Quiz

## Purpose
Funnel to the test. This is the MOST direct recruit-to-test pillar. Everything else builds brand; this pillar converts.

## Format
- M4 (Interactive / Quiz) primary
- IG Stories (sticker polls, quizzes) — highest engagement
- IG Feed (static multi-choice visual)
- LinkedIn (limited — native polls only)

## Content types

### Type E1 — "Which X are you?"
- Quote picker → type hint
- Scenario picker → type hint
- Preference picker → type hint

### Type E2 — "Complete the sentence"
- "My worst fear is..." → options each map to 1 type's core fear
- "When I'm stressed I..." → options map to stress responses

### Type E3 — "Spot the type"
- Describe a character/scenario → which type
- "Which Friends character is a Type 2?" — pop culture hook

### Type E4 — "Rank these"
- "Rank these 4 values in order" → pattern reveals primary type

## Critical design rules

### Options must DIFFERENTIATE
Bad: "A: I like helping / B: I help sometimes" — both lead nowhere
Good: "A: I notice who's upset / B: I solve the problem on the table / C: I leave the room" — each maps to different types

### Never FORCE a single type
Always CTA: "This hints at Type X — take the full test to confirm"

### Keep it short
Max 4 options. Max 1 question per post. Complexity kills engagement.

## Mix for 30-day calendar
- 15% of IG content
- 5% of LinkedIn
- Weekly Story quizzes (3-5 stickers/week)

## Hook patterns
- "Pick the quote that hits hardest:"
- "Your default in a group is:"
- "Which one lands for you?"
- "Comment A, B, C, or D:"

## Do
- Invite comment engagement
- Reply to commenters with type hints
- Follow up: next week's post references "many of you picked A — you might be a Type X"
- Use ambiguity (several options could fit) — readers who want clarity take the test

## Don't
- Use this pillar to "type" someone definitively
- Make options boring or lopsided
- Skip the "take the test" CTA
- Run same quiz format weekly — rotate E1/E2/E3/E4
