# Enneagram Glossary — Preferred English Terms

Use these exact EN terms. Never translate to PT in the EN caption.

## Core terms

| Term | Definition | Common mistake |
|---|---|---|
| **Type** (or Ennea-type) | One of the 9 core personality structures | "kind", "category", "profile" — use **type** |
| **Wing** | The adjacent type that colors the core (e.g., "4w3", "4w5") | "influence" — use **wing** |
| **Instinct** (or instinctual variant / subtype) | SP (Self-Preservation), SX (Sexual / One-on-One), SO (Social) | "focus", "drive" — use **instinct** |
| **Fixation** | The mental/cognitive pattern of each type | "obsession" — use **fixation** |
| **Passion** | The emotional engine of each type (one of the 9 classical passions) | "feeling" — use **passion** |
| **Virtue** | The healthy counterpart to each passion | "trait" — use **virtue** |
| **Holy Idea** | The spiritual/objective insight restored at high health (Naranjo/Ichazo) | — |
| **Center** (triad) | Head (5-6-7), Heart (2-3-4), Body/Gut (8-9-1) | "group", "category" |
| **Health level** | Riso & Hudson's 1-9 scale (1 best → 9 worst) | "stage" |
| **Integration** (Direction of growth) | The arrow of healthy movement (e.g., 4→1) | "evolution" |
| **Disintegration** (Direction of stress) | The arrow under pressure (e.g., 4→2) | "regression" |
| **Line** | Either integration or disintegration line connecting types | — |

## Phrases we use vs avoid

| ✅ Use | ❌ Avoid |
|---|---|
| "For Fours..." | "If you're a 4..." (digit) in body copy |
| "Type 4 (the Individualist)" on first mention | "Type Four" in a post where we use "Type 3" elsewhere — pick digit or word and stay consistent |
| "Countertype" (Chestnut/Naranjo concept) | "Opposite" |
| "Tritype" (only when explaining) | — |
| "Arrow" (informal) or "Line" (formal) | "Link" |
