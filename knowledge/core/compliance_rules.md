# Compliance Checklist (auto-validated)

Every post must pass these checks before publish.

## hasCitation
✓ Post contains exactly one citation with attribution (author + work + year)
✗ Post has no quoted source or generic "research shows..."

## citationHasPage
✓ Citation includes page number ("p.157")
✗ Citation is from the 56% of acervo without page — flag warning, allow with override

## hedgingLanguage (when topic requires)
✓ No absolute claims about individuals
✓ "reads as" / "may resonate" for celebrities
✓ "The Enneagram is a framework, not a diagnosis" when mental health is mentioned

## withinHashtagLimit
✓ IG: 15-25 hashtags
✓ LinkedIn: 3-5 hashtags

## altText present
✓ 1-2 sentence English description for accessibility + SEO

## No prohibited content
✗ Medical claims ("cure", "treat", "heal")
✗ Specific diagnoses (depression, ADHD, autism) attributed to a type
✗ Political stances
✗ Religious claims (except quoting author if author made one)
✗ Investment, financial, or legal advice
✗ Body-shaming / weight-loss claims
✗ Before/after testimonial style

## Required disclaimers (context-dependent)

### Mental health context
Append at end of caption:
> *The Enneagram is a personality framework, not a medical diagnosis. If you're struggling, please reach out to a licensed professional.*

### Celebrity context
Append at end:
> *Typing public figures is inherently speculative. This reflects publicly available material and common community consensus — not a professional assessment.*

### Relationships / compatibility context
Append at end:
> *Type compatibility is a lens, not a rule. Healthy relationships happen across every type pairing.*

## Source attribution — NEVER allowed
- Quoting an author without naming them
- Paraphrasing content AS IF it were a citation ("As research shows...")
- Citing authors we don't have in our acervo (use only the 9 approved)
- Page numbers we don't have (if page = 0, render without page; don't invent)
