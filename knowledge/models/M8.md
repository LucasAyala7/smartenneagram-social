# M8 — Reel / Short Video

## When to use
- Any pillar (algorithmic priority)
- IG Reels (primary)
- LinkedIn short video (emerging format)
- 15-30 seconds ideal; never over 60s

## Format
- Vertical 1080x1920
- First 2 seconds determine 80% of retention — hook must be IMMEDIATE

## Output deliverables (what the app produces)
1. **Cover / thumbnail prompt** (for the still image that appears in feed)
2. **Script in timed blocks** (0-3s, 3-10s, 10-25s, 25-30s)
3. **B-roll suggestions** (what visuals go with each block)
4. **On-screen text overlays** (per block)
5. **Suggested audio mood** (not specific tracks — we can't license)

## Script template

```
[0-3s HOOK]
On-screen: "[Punchy 4-7 word hook]"
Voiceover: "[same or variant]"
B-roll: [fast, attention-catching visual]

[3-10s SETUP]
Voiceover: "[the framing — the insight]"
On-screen: keyword highlights
B-roll: [relevant people/scenes]

[10-25s PAYOFF + CITATION]
Voiceover: "[the core idea + a citation, paraphrased or direct]"
On-screen: citation block appears — "Author (Year, p.X)"
B-roll: [slower, more emotional]

[25-30s CTA]
Voiceover: "[soft CTA]"
On-screen: "Take the test — link in bio"
B-roll: brand frame or outro
```

## Hook patterns that retain

- **Pattern interrupt**: "You're not a Four. You're probably a Six."
- **Curiosity gap**: "There's one line in the Enneagram that most Fours miss."
- **Contrarian**: "Forget what you've heard about Type 7s."
- **Stat**: "Only 4% of Fours have this pattern..."
- **Direct**: "If you hide your pain in productivity, you might be a Three."

## Visual rules
- ONE subject per cut when possible
- Cuts every 1.5-3 seconds
- Text overlays in the top-third or middle (avoid bottom — UI clutters)
- Captions baked-in (accessibility + sound-off viewing)
- Brand watermark subtle (opacity 50%) in the corner

## Image prompt (cover)
- First frame of the video = the thumbnail
- Make it count: subject + emotion + visible text hint of what's coming
- Leave room for "MORE" button / caption truncation

## Copy-side strategy (caption for the reel)
- The reel carries the main content
- Caption = 1 hook line + 1 context line + CTA + hashtags
- Don't dump the full script in caption

## Common mistakes
- ❌ Talking head 30s with no cuts
- ❌ Slow intro "Hey guys, today I want to talk about..."
- ❌ Citation shown too fast to read (min 2s dwell)
- ❌ Caption-off viewing impossible (no on-screen text)
- ❌ Generic stock b-roll that doesn't match the emotion
