# M3 — Carousel Deep-Dive

## When to use
- Pillar C (Type in Context — deep exploration of one angle)
- Pillar F (Behind the Science — methodology, Big Five, research)
- Pillar D (Comparison — side-by-side types)
- Instagram Feed (carousels get highest reach)
- LinkedIn (document posts equivalent)

## Format
- 6-10 slides
- Slide 1 = cover (hook), Slides 2-N = 1 idea each, Slide N = CTA

## Slide-by-slide discipline

### Slide 1 — Cover
- Strong headline (same as caption hook)
- Visual pulls scroll-stop
- Small "swipe →" cue bottom-right (subtle)
- No detail — promise only

### Slides 2-N — Body (the "ideas")
Pick ONE of these structures and stay consistent:
- **Structure A — listicle**: "1/7. Fours are the only type in the Heart Triad who...", "2/7..."
- **Structure B — scene + concept**: each slide is a behavior on top, concept below
- **Structure C — citation → explanation**: each slide opens with a quote, then unpacks
- Each slide = 1 headline + 2-3 lines of body
- Never dump paragraphs — if it doesn't breathe, it won't be read

### Slide N-1 — The Citation
- One full slide for the anchor citation
- Formatted like a block quote
- Author + work + year + page prominent

### Slide N — CTA
- "Want to find your type with citations like this one?"
- "Take the free test at SmartEnneagram — link in bio"
- "Save + share with someone who needs to hear this"

## Visual consistency across slides
- Same palette, same fonts, same margins
- Small page indicator (1/8, 2/8)
- Logo present on cover and CTA only, not on middle slides (less busy)

## Image prompt guidance
- Prompts per slide (one per slide if imagery used)
- Middle slides can be typography-only (cleaner)
- Cover needs strongest visual

## Copy-side strategy
- Caption LESS work (the carousel does it)
- Hook repeats the cover
- Rest of caption: slight expansion + CTA + hashtags

## Common mistakes
- ❌ 20 slides (engagement drops after 7-8)
- ❌ Paragraph-dump slides
- ❌ Cover that reveals the payoff
- ❌ Citation as afterthought — it should have its own slide
